S.U.N. Sistema Universal de Notas

Software para colegios con gestión de notas, matrículas y asistencias. 68% Blade, 35+ tablas, auditoría, roles, gráficas y chatbot. Desarrollado con Laravel, Vite, Node.js y MySQL.

Ver repositorio

Resumen

Desarrollé un sistema integral para un colegio que necesitaba automatizar el registro y control de notas, matrículas y asistencias. El software permite gestionar roles como: Secretario, Rectoría, Superadmin, Secretaria-Aux, Alumno y Docente.

Características principales

  • Gestión de roles con permisos diferenciados
  • Módulo de asignación de profesores a materias y cursos
  • Subida de notas por periodos con activación controlada
  • Gráficas interactivas para alumnos (visualización de progreso)
  • Chatbot integrado (API externa)
  • Auditoría media de cambios (quién, cuándo, qué modificó)
  • Validación de correo y notificaciones
  • 35+ tablas relacionadas con estructura MVC y POO
  • 68% Blade — para mantener consistencia en la interfaz y evitar caídas del frontend

Tecnologías utilizadas

Laravel PHP MySQL Blade JavaScript Vite Node.js AJAX Auth Git

¿Cómo funciona?

El sistema permite a los docentes subir notas, a los secretarios gestionar matrículas, a la rectoría ver reportes, y a los alumnos visualizar su progreso con gráficas. Todo con una interfaz responsive y segura.
